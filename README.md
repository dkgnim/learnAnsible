# Изучение Ansible.
***
В процессе изучения ознакомился с принципом работы Ansible.
По заданию требовалось создать playbook для автоматизации сборки и запуска приложения Boxfuse HelloApp на двух slave серверах:
* Первый сервер (buildServer). Установка maven и git. Клонирование с github и сборка Java приложения при помощи Maven и синхронизация артефакта с сервером prodServer.
* Второй сервер (prodServer). Установка tomcat8 (при его отсутствии) и запуск Boxfuse HelloApp.
